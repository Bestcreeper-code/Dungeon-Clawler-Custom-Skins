Value = CalculatedValue

CalculatedValue are dynamic values that are recalculated each time an ability using them is called

values that have a # before their name mean that that value is one of the lines in the #list at the end of the file 

ex for IfElse{#Comparison_Types, Value  Value, Value  CompareValue, Value  ValueWhenTrue, Value  ValueWhenFAlse}:

IfElse{GreaterThan, CountOfItemsInDeck{}, 20, 100, Divide{CurrencyAmount{}, 30}}

gives 100 if you have more than 20 items and gives your amount of gold/30 if you have 20 items or less
